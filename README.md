# NitroPerks插件
功能:
* **客户端GIF头像与横条**
* **1080P 60FPS投屏**;
* **随处使用跨服务器和动画表情符号**.


# 教程 
1. 下载 [Better Discord](https://betterdiscord.app);
2. 安装好后, 打开 **设置**, 下滑并找到 **插件** 选项并点击 **打开插件文件夹**;
3. 拖入下载好的**NitroPerks**插件;
4. 打开插件.
